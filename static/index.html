DUMPTRUCK

Dumptruck for Lamdba is a npm package for tailing lambda logs in (near) real time. It aims to take the hard work out of setting up cloudwatch logging for your Lambda projects, and will automatically load config from your AWS config and integrates with serverless framework projects to streamline the logging process.

To get started, just install via NPM:

npm install dumptruck

Once installed, just follow the prompts to see your cloudwatch logs
